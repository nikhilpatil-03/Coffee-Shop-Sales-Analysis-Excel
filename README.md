Coffee Shop Sales Analysis (Excel Project)

Project Overview
This project focuses on analyzing coffee shop sales data using Microsoft Excel. The goal was to transform raw sales data into meaningful business insights and build a dashboard for decision-making.

Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization
- Dashboard Creation
- Excel Formulas

Objectives
- Analyze sales performance
- Identify top-selling products
- Study monthly sales trends
- Understand customer purchasing patterns

Key Insights
- Identified peak sales periods.
- Highlighted high-performing products.
- Created interactive dashboard for quick analysis.

Skills Demonstrated
- Data Analysis
- Business Insight Generation
- Dashboard Design
- Excel Data Handling
